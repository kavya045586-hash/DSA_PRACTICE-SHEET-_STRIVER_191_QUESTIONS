This repository contains my daily practice solutions for the **Striver DSA Sheet (191 Questions)**.  
The goal is to strengthen Data Structures and Algorithms (DSA) concepts and prepare for coding interviews and campus placements.

---

## 📌 About the Sheet
The Striver DSA Sheet is a curated set of **191 important coding questions** covering:
- Arrays
- Strings
- Recursion
- Linked List
- Stack & Queue
- Binary Trees
- Binary Search Trees
- Graphs
- Dynamic Programming
- Greedy Algorithms
- And more…

It is widely recommended for interview preparation and competitive programming.

---

## 🛠️ Repository Structure
- Each question is solved in **C++** (sometimes other languages if explored).
- Solutions are organized by **topic folders** (e.g., Arrays, Strings, Trees).
- Each file contains:
  - Problem statement (as comments)
  - Solution code
  - Example input/output
